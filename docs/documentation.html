LEANWI Links Manager Documentation

Welcome to the LEANWI Link Manager plugin documentation.
This guide will help you configure and use the plugin efficiently.

Features Include:

  • Content providers will be able to add new links in one place, adding attributes and tags to those links making the links automatically searchable and automatically show up on the website in the correct location.
  • Visitors to your site will have the ability to search and filter your list of links by Program Area, Format, Tags and Keywords.
  • Three different ways to display your links: Table, Feed and Featured (3 column format)
  • Once set up, should allow for easy addition and removal of links from your website. Add/Remove your link and then navigate to the page and check it to verify everything has been done correctly.
  • Your links pages will have consistent formatting and usage across your website for better visitor usability

Administrators:

  • All attributes are configurable and can either be pre setup if you know what attributes your links should have or made as needed.
  • Link Listings are shown on pages/posts/articles etc. via a simple configurable versatile shortcode.
